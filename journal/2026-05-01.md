## Lessons / patterns

- **Eight bugs caught in dogfood across the day** all surfaced via running the experiment feature against a real codebase (TradingAgents). Each bug → a 5-15 min fix → a tagged release. The cycle worked.
- **Plan-reviewer's third pass on AGENTSKILL** caught real corner cases (envelope shape drift between skills, abort test buried under "improvements", audit-log file fragmentation). Worth running it more than twice when stakes warrant.
- **Single-source-of-truth refactors** — `lib/{review,pentest,score}-service.js` are the right pattern. The route becomes a thin SSE/HTTP wrapper; the agent skill becomes a thin envelope wrapper; both call the same validate-then-call-Ollama function. Drift between agent and UI paths is now structurally prevented.
- **Comma-separated chip input** — small UX fix that mattered when users paste multi-value strings. Worth checking other chip-style inputs (paths, tags) for the same pattern.
