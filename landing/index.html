Features

Code Report Cards

Get an honest, graded assessment of your code — security, bugs, performance, and maintainability all scored and explained clearly.

Plain English Explanations

No jargon, no condescension. Just friendly, clear explanations with helpful analogies when you need them.

Copy-Paste Fix Prompts

Every issue comes with a ready-to-use prompt you can paste into your AI coding tool for instant fixes.

Local AI — Your Code Stays Private

Powered by Ollama running on your machine. Your code never leaves your computer. No cloud, no tracking, no subscriptions.

11 Modes for Every Task

Chat, Explain, Safety Check, Clean Up, Code to Plain English, Idea to Spec, Review, Create, Prompting, Skillz, and Agentic modes cover every workflow.

Built for Non-Technical Users

Designed from the ground up for people who use AI to build things — not for developers. Friendly, patient, and helpful.

First Launch Instructions

macOS: Right-click the app and select "Open" (not double-click). Click "Open" in the warning dialog.
Windows: Click "More info" in the SmartScreen warning, then click "Run anyway".
Linux: Make the AppImage executable: chmod +x Code-Companion.AppImage
