   - Import `autoUpdater` from `electron-updater`
   - Import `electron-log` for logging
   - Import `createBackup` from `./data-manager.js`
   - `initAutoUpdater(win, dataDir)`:
     - Set `autoUpdater.logger = log`
     - `autoUpdater.checkForUpdatesAndNotify()` — called once on startup
     - `autoUpdater.on('update-available', (info) => { win.webContents.send('update-available', info); })`
     - `autoUpdater.on('update-downloaded', async (info) => {`:
       a. Call `await createBackup(dataDir)` — pre-update backup per user decision
       b. `win.webContents.send('update-downloaded', info)`
     - Register IPC handler: `check-for-updates` -> `autoUpdater.checkForUpdatesAndNotify()`
     - Register IPC handler: `restart-for-update` -> `autoUpdater.quitAndInstall()`

3. Update `electron/main.js`:
   - Import `initAutoUpdater` from `./updater.js`
   - Call `initAutoUpdater(win, dataDir)` after window is created and shown

4. Update `electron/preload.js` — add auto-update IPC:

   - `restartForUpdate: () => ipcRenderer.invoke('restart-for-update')`

   - `onUpdateDownloaded: (callback) => ipcRenderer.on('update-downloaded', (_, info) => callback(info))`

     - `"electron:build:win": "npm run build && electron-builder --win --config electron-builder.config.js"`
     - `"electron:build:linux": "npm run build && electron-builder --linux --config electron-builder.config.js"`
