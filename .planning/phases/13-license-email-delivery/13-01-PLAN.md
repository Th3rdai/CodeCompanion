# Phase 13: License Email Delivery — Plan

**Phase:** 13-license-email-delivery
**Type:** execute
**Depends on:** Phase 9 (can run in parallel with 10)

## Objective

Send license keys to customers via email with a professional, branded template.

## Tasks

### Task 1: Email sender module

**File:** lib/email-sender.js (new)

1. Export `sendLicenseKey({ to, key, features, expires })`
2. Support providers: SendGrid, Resend, or nodemailer (SMTP)
3. Config: `EMAIL_PROVIDER`, `EMAIL_API_KEY`, `EMAIL_FROM`
4. Load HTML template, substitute `{{key}}`, `{{features}}`, `{{expires}}`
5. Subject: e.g. "Your Code Companion Pro License"
6. On failure: log, throw or return error for caller to handle

### Task 2: Email template

**File:** templates/license-email.html (new)

1. HTML email with inline CSS
2. Placeholders: `{{key}}`, `{{features}}`, `{{expires}}`, `{{supportUrl}}`
3. Copy-paste friendly key display (monospace, selectable)
4. Instructions: paste into Settings → License
5. Support link, branding

### Task 3: Retry logic

1. On send failure: retry up to 3 times with backoff
2. Log failures for manual follow-up
3. Optional: dead-letter queue or log file for failed sends

### Task 4: Integration

1. Phase 10 webhook calls `sendLicenseKey` after generating key
2. Phase 9 batch mode: optional `--email` flag to send each key (requires config)
3. Document env vars in README

## Verification

- [ ] Test send to real inbox receives key
- [ ] Template renders correctly in Gmail, Outlook
- [ ] Invalid API key fails gracefully with clear error
